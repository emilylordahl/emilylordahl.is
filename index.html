SpiritGuide

SpiritGuide is a web application that will lead users to yoga studios, vegetarian restaurants, juice bars and farmers markets by searching one of the five boroughs of New York. Built with Ruby on Rails and Backbone.js, data was parsed and rendered via Yelp's API and NYC OpenData. Location data for businesses is displayed via the Google Maps API. Search results saved to a user's profile populated a PostgreSQL database using a Rails server.

CheckUp

CheckUp is a single-page web application, built with Backbone.js, that allows a user to securely sign up for an account and login to receive health tips based on his or her age and gender. A user can mark certain health tips as complete to track yearly checkups and maintain a healthy lifestyle. Data was parsed and rendered via the HealthFinder.gov API. Users and their health progress were collected in a PostgreSQL database using Sequelize on a Node.js server.

featur

featur is a single-page web application, built with Backbone.js, that allows a user to search for a music artist to find collaborators and their shared songs. Users can create and save personal playlists and playback songs through Spotify's web plugin. Data was parsed and displayed via Spotify's API. Playlists and songs were stored in a PostgreSQL database using Sequelize on a Node.js server. Built with Eric Kwong and John Sanders.

about

Montréal born, Virginia raised, Brooklyn livin’.

I completed my B.F.A. in Graphic Design & Art History from James Madison University in 2011 and worked as a print designer in the years following. As such, white space, grid systems and beautiful typography will always have my heart.

In 2015, I got the itch to step up my design skills and completed (as well as taught) General Assembly's Web Development Immersive, where I learned the fundamentals of programmatic thinking. Since then, I've combined the two worlds of design and development by working as the Marketing Engineer at Conversocial and am transitioning into a role as the UX Developer.

If I'm not designing or brushing up on my coding capabilities, you can find me traveling, in a coffee shop reading, spending time with friends and family or squealing at the sight of basically any dog.
